JS 作用域有哪些：
    1. 全局作用域
    2. 函数作用域
    3. 块级作用域

全局作用域：
    1. 全局变量：在全局作用域中定义的变量，在整个程序中都可以访问。
    2. 全局对象：在浏览器环境中，全局对象是 window 对象。在 Node.js 环境中，全局对象是 global 对象。
函数作用域：
    1. 函数作用域：在函数内部定义的变量，只能在函数内部访问。
    2. 函数作用域链：当函数嵌套时，内层函数可以访问外层函数的变量，形成作用域链。
块级作用域：
    1. 块级作用域：在块级作用域中定义的变量，只能在该块级作用域中访问。
    2. let 关键字：在块级作用域中使用 let 关键字定义的变量，只能在该块级作用域中访问。
    3. const 关键字：在块级作用域中使用 const 关键字定义的变量，只能在该块级作用域中访问。
